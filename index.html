The Four Garden Sub-Flows

All four intelligent flows live inside one combined n8n workflow — tending, tracking, searching, and connecting your reading world.

📖
Sub-Flow 1 · Acquisition

Multilingual Librarian

Add any book in English, Urdu, or Arabic. AI auto-detects language, assigns genre, picks a plant spirit, and fetches the cover — all for free.

  • Google Books + Open Library cover fetch
  • Google Gemini AI genre & plant spirit classification
  • Auto language detection (no paid API)
  • Saves to Airtable + Google Sheets
  • Discord beautiful embed notification
📋 See Setup Guide

Sub-Flow 2 · Inventory

Loan Manager

Issue and return books with a form. Weather checks Karachi weather daily and suggests the perfect genre to read today.

  • Issue / Return form with Airtable update
  • Google Calendar + Tasks reminders
  • OpenWeatherMap → daily genre suggestion
  • Gmail overdue alerts with beautiful HTML
  • Discord daily reading suggestion
📋 See Setup Guide

Sub-Flow 3 · Engagement

Semantic Search & Archive

Search any character, quote, or topic across your entire library. Completed books are auto-archived to GitHub with AI-written summaries.

  • AI-powered search across quotes & notes
  • Google Drive PDF search included
  • Results emailed instantly via Gmail
  • Daily "Water Your Mind" quote via Gmail + Discord

Setup Guide — Import the Combined Workflow

One file, one import, all four sub-flows active. Follow these steps on your Railway n8n instance.

⚡

One Combined Workflow File

The entire Bibliophile Concierge system — all four sub-flows, all Airtable mappings, all AI nodes — is packaged in a single importable JSON. No separate files needed.

⬇️ Download Workflow JSON 🔗 View Live on Railway 🐙 GitHub Repo
1

Import the Combined Workflow into n8n

In your n8n instance → Workflows → ⋯ menu → Import from URL, paste the raw GitHub URL below. Or download the JSON and use Import from File.

https://raw.githubusercontent.com/wardha-netizen/bibliophile-concierge/main/bibliophile-concierge-complete.json
2

Connect Google OAuth (covers Gmail · Sheets · Drive · Calendar · Tasks)

In n8n → Credentials → New → Google OAuth2. One credential covers all Google services. Grant all required scopes when prompted.

3

Add Airtable Personal Access Token

Visit airtable.com/create/tokens → create a token with data.records:read and data.records:write scopes on your Bibliophile base. Paste it into the Airtable credential in n8n.

Base ID: apptGlLDt9VHCE9ey  ·  Table: Books (tblENLBbFVBYaEahV)
4

Configure Groq AI as OpenAI-Compatible

In n8n → Credentials → New → OpenAI. Set the base URL to Groq's endpoint and paste your free Groq API key from console.groq.com.

Base URL: https://api.groq.com/openai/v1  ·  Model: llama-3.3-70b-versatile
5

Add GitHub Personal Access Token

GitHub → Settings → Developer Settings → Fine-grained PATs → create a token with Contents: Read & Write on the bibliophile-concierge repo. Add it as a GitHub credential in n8n.

6

Set Up Airtable — Books Table Schema

Create a base called Bibliophile with a Books table. Required fields: Book Title, Author, Language, Genre, Summary, ISBN, Publisher, Publication Date, Edition, Number of Pages, Cover Image, Lending Status, Added By, Format, Price, Rating, Tags, Availability Worldwide, Book Condition, Is Archived, Motivational Quotes, Related Events, Attachment Summary.

7

Activate the Workflow & Test

Toggle the workflow Active in n8n. The webhook URLs for the Book Intake Form, Loan Form, Search Form, and Book Request Form will appear on the trigger nodes. Share those URLs as your library intake forms.
