The Four Garden Workflows

Each workflow is a living part of the library — tending, tracking, searching, and connecting your reading world.

📖
Workflow 1 · Acquisition

Multilingual Librarian

Add any book in English, Urdu, or Arabic. AI auto-detects language, assigns genre, picks a plant spirit, and fetches the cover — all for free.

  • Google Books + Open Library cover fetch
  • Groq AI genre & plant spirit classification
  • Auto language detection (no paid API)
  • Saves to Airtable + Google Sheets
  • Discord beautiful embed notification
📋 Import Workflow 1
📊
Workflow 2 · Inventory

Loan Manager

Issue and return books with a form. Weather checks Karachi weather daily and suggests the perfect genre to read today.

  • Issue / Return form with Airtable update
  • Google Calendar + Tasks reminders
  • OpenWeatherMap → daily genre suggestion
  • Gmail overdue alerts with beautiful HTML
  • Discord daily reading suggestion
📋 Import Workflow 2
🔍
Workflow 3 · Engagement

Semantic Search & Archive

Search any character, quote, or topic across your entire library. Completed books are auto-archived to GitHub with AI-written summaries.

  • AI-powered search across quotes & notes
  • Google Drive PDF search included
  • Results emailed instantly
  • Daily "Water Your Mind" quote via Gmail + Discord
  • Auto-archive to GitHub + Google Sheets
📋 Import Workflow 3
📱
Workflow 4 · Team Hub

WhatsApp Collaboration

Request books, get daily team digests, and receive instant return alerts — all 3 team members notified via WhatsApp Cloud API.

  • Book request form → WhatsApp all 3 members
  • Daily 9AM digest: stats + who has what
  • Return alerts: "book available now!"
  • All requests logged to Google Sheets
  • Works in WhatsApp testing mode
📋 Import Workflow 4
