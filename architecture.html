The Four Sub-Flows

All four live inside one n8n workflow (ID: M25bDNquM81jUjQh) — 74 nodes, 4 form triggers, 5 scheduled crons, sharing the same Airtable base and Google credentials.

📖
Sub-Flow 1

Acquisition & Multilingual Librarian

📋 n8n Form Trigger
User submits book (English / Urdu / Arabic)
↓
📚 Google Books API
Fetch metadata, cover thumbnail
🏛️ Open Library API
Parallel cover fallback via ISBN
↓
🌐 Language Detection
Unicode range check (no paid API)
↓
🔮 Google Gemini AI (3 nodes)
Genre classification + plant spirit assignment
↓
🌱 Airtable — Books Table
Create record with all 20+ fields
📊 Google Sheets — Bibliophile Vault
Append backup row
↓
💬 Discord Webhook
Rich embed — new book planted 🌱
📊
Sub-Flow 2

Inventory & Loan Manager

📋 n8n Loan Form
Issue or Return action selected
↓
🗃️ Airtable — Update Lending Status
Available ↔ Issued
📅 Google Calendar
Tentative return-date event created
✅ Google Tasks
Task reminder created for borrower
📊 Google Sheets — Loans Log
Append loan transaction row
⏰ Daily 9AM Cron
Automatic weather-based book suggestion
↓
🌤️ OpenWeatherMap — Karachi
Current conditions → genre mapping
🌸 Pick Available Book
Airtable query filtered by genre
💬 Discord Daily Suggestion
"Rainy day? Read some Poetry 📖"
⏰ Daily Overdue Check
Scans all Issued books past due date
↓
📧 Gmail — Overdue Alert
HTML email to borrower
💬 Discord Ping
Team notification with list of overdue books

Sub-Flow 3

Engagement & Semantic Search

🌅 Daily 8AM Cron
"Water Your Mind" motivational quote
↓
🤖 Groq AI — Generate Quote
Unique daily literary quote
📧 Gmail + 💬 Discord
Beautifully formatted daily quote
🔍 Book Search Form
User types quote / character / topic
↓
🗃️ Airtable Semantic Search
Searches Notes, Summary, Title (parallel)
📁 Google Drive PDF Search
Full-text search across uploaded PDFs
↓
🤖 Groq AI — Merge & Rank
LLM consolidates and scores results
📧 Gmail — Results Email
Formatted results delivered instantly
🌙 Evening Archive Cron
Finds Airtable books marked "Completed"
↓
🤖 Groq AI — Generate Summary
AI-written book summary + tags
▶️ YouTube Search API
Find related book discussion video
🐙 GitHub — Push Markdown
Archive_YYYY-MM-DD.md committed
📒 Google Sheets — Archive Log
AI Summary + GitHub link + YouTube
✅ Airtable — Mark Archived
Is Archived checkbox set to true
